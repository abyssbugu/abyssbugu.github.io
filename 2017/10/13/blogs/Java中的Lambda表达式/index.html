Java中的Lambda表达式

发表于 2017-10-13 | 分类于 Java

背景介绍

Lambda表达式已经使用有一段时间了,这次专门整理一下Lambda表达式笔记

1. 什么是lambda?

Lambda（λ）是一個希腊字母（Λ 是它的大写字母），用來表示许多观念：

  1. 物理中用來表示波長的符号
  2. 数学中用來表示空字串的符号
  3. 计算机编程中用來表示匿名函数（Anonymous Function）的符号.

2. 为什么使用lambda表达式?

主要有三个原因：

  1. 更加紧凑的代码 比如Java中现有的匿名内部类以及监听器(listeners)和事件处理器(handlers）等都显得很冗长 ,使用lambda进行简化
  2. 修改方法的能力,说白了，就是函数中可以接受以函数为单元的参数 例如在Java中使用集合类对一个字符串按字典序列进行排序，代码如下所示:

是不是很方便，我们不需要实现Comparable接口，使用一个Lambda表达式就可以改变一个函数的形为~

  1. 更好地支持多核处理 例如，通过Java8新增的Lambda表达式，我们可以很方便地并行操作大集合，充分发挥多核CPU的潜能。并行处理函数如filter、map和reduce。
